Chronic consumption, craving, tolerance, withdrawal symptoms, and loss of control over drinking. This assessment uses the AUDIT (Alcohol Use Disorders Identification Test) to measure risky consumption and dependence severity.

Nicotine

NUD · FTND

Compulsive tobacco use, withdrawal (irritability, anxiety, craving), tolerance, and failed cessation attempts. Based on Fagerström Test for Nicotine Dependence (FTND) plus DSM‑5 criteria.

Cocaine

CocUD · DSM‑5

Binge patterns, intense craving, paranoia, financial/social collapse, and physiological toxicity. Uses modified Cocaine Selective Severity Assessment (CSSA) and DSM‑5 criteria.

⚕️ Start cocaine assessment →

Gambling

ICD‑11 6C50

Progressive betting, chasing losses, deception, and financial consequences. Based on PGSI (Problem Gambling Severity Index) and DSM‑5 criteria for gambling disorder.

⚕️ Start gambling assessment →

Caffeine

CUD (proposed)

Physiological dependence, insomnia, anxiety, withdrawal headaches, and tolerance. Screener includes intake patterns, sleep disruption, and rebound fatigue.

Codeine

OUD · DSM‑5

Opioid use disorder criteria: escalating doses, withdrawal (flu-like symptoms), tolerance, and compulsive use despite harms. Assessment leverages Current Opioid Misuse Measure (COMM) and DSM‑5 indicators.

Cannabis

CUD · DSM‑5

Amotivational syndrome, memory deficits, increased tolerance, and withdrawal irritability. Uses CUDIT‑R (Cannabis Use Disorders Identification Test).
